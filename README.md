# UFMFUT-15-3-Human-Robot-Interaction-Technologies

Official repository for the human robot interaction technologies (BSc module) at University of the West of England (UFMFUT-15-3)

## Description

This repository contains code examples, tutorials, and resources for the Human-Robot Interaction Technologies module. The materials cover various aspects of human-robot interaction, including chatbots, speech recognition, and robot programming.
