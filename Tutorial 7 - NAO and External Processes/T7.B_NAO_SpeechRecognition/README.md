# Speech Recognition with NAO and External Python Processes

Scripts for using a Python Box to communicate with external Python scripts from Choregraphe.

First use GetDeviceIdx.py to get the index of your current microphone.

Then run first the NLPServer.py in terminal, then the Choreographe node.

Original from [Paul Bremner at https://github.com/PaulBremner/Choregraphe/tree/master](https://github.com/PaulBremner/Choregraphe/tree/master)
